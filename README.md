# Practical Libertarianism

Practical Libertarianism is a long-form, constitutionally grounded policy manuscript supported by research papers and implementation proposals.

## Project Goals

- Produce a publishable manuscript organized into five core parts.
- Support each claim with verifiable legal, historical, and policy sources.
- Pair high-level arguments with concrete implementation proposals.

## Repository Structure

```text
/
├── Manifesto Chapters/      # Reader-facing manuscript content
├── Research Papers/         # Supporting legal/historical/economic research
├── Proposals/               # Constitutional, legislative, and transition proposals
├── docs/                    # Working records, planning, and decision artifacts
├── references/              # Canonical reference material
├── assets/                  # Diagrams and media
├── PROJECT_REQUIREMENTS_SPECIFICATION.md
├── MASTER_ARCHITECTURE.md
├── STYLE_GUIDE.md
└── WORK_ITEMS.md
```

## Working Documents

- `MASTER_ARCHITECTURE.md`: mission, standards, and quality gates.
- `STYLE_GUIDE.md`: editorial and structural writing standards.
- `PROJECT_REQUIREMENTS_SPECIFICATION.md`: requirements and status checklists.
- `CHANGELOG.md`: notable repository changes.

## Editorial Workflow

1. Research
2. Outline
3. Draft
4. Constitutional review
5. Technical/fact review
6. Citation completion
7. Editorial review
8. Publish
