# Project Style Guide

## Purpose

This guide defines the engineering and editorial standards for the
Practical Libertarianism project.

## Writing Principles

-   Write to persuade through reason, not rhetoric.
-   Separate facts from recommendations.
-   Acknowledge reasonable counterarguments.
-   Preserve constitutional institutions where possible.
-   Prefer precise language over emotionally loaded language.

## Chapter Template

1.  Executive Summary
2.  Constitutional Framework
3.  Historical Background
4.  Current Law
5.  Problem Statement
6.  Proposed Reform
7.  Benefits
8.  Risks / Counterarguments
9.  Implementation
10. Conclusion
11. References

## Source Standards

-   Primary sources preferred.
-   Cite Constitution, statutes, court opinions, historical records.
-   Clearly distinguish opinion from fact.

## Proposal Standards

Each proposal identifies: - Required authority - State or Federal
action - Constitutional amendment required? - Estimated implementation
difficulty - Transition plan

## Repository Conventions

Manifesto Chapters = reader-facing. Research Papers = supporting
evidence. Proposals = implementation documents.
