# Document Inbox

Place newly recovered or generated material here only when its permanent
location is not yet clear.

For each item:

1. Review and classify it as manuscript, research, proposal, reference, or
   project documentation.
2. Register it in `PROJECT_REQUIREMENTS_SPECIFICATION.md` or link it to an
   existing registered item.
3. Move it to the appropriate permanent directory.

The inbox is a temporary intake location, not a long-term archive.
